# Ctrl+Enter Sender

Enterキーの誤送信を防止するChrome拡張機能です。  
送信は **Ctrl+Enter**（Macは **Cmd+Enter**）のみ、素のEnterは改行になります。

## セキュリティ

- **外部通信一切なし** — ネットワークリクエストを送信しません
- **最小権限** — `storage`（設定保存）と `activeTab`（現在のタブのURL取得）のみ
- **コード完全公開** — すべてのソースを自分で確認できます
- **自動更新なし** — ローカルインストールなのでコードが勝手に変わりません

## インストール方法

1. このフォルダをPCの任意の場所に保存
2. Chromeで `chrome://extensions` を開く
3. 右上の「**デベロッパーモード**」をONにする
4. 「**パッケージ化されていない拡張機能を読み込む**」をクリック
5. このフォルダ（`ctrl-enter-sender`）を選択
6. 完了！アドレスバー横にアイコンが表示されます

## 使い方

- **アイコンクリック** → ポップアップが開く
- **現在のサイト** → そのサイトでのON/OFFを切り替え
- **新しいサイトでデフォルトON** → 初めて訪問するサイトでの動作を設定
- **サイト個別設定** → サイトごとの設定一覧と管理

## 動作仕様

| キー操作 | 有効時の動作 |
|---------|------------|
| Enter | **改行**（送信しない） |
| Ctrl+Enter / Cmd+Enter | **送信** |
| Shift+Enter | **改行**（従来通り） |

## 対応確認済みサイト

- ChatGPT (chatgpt.com)
- Claude (claude.ai)
- Slack (app.slack.com)
- Discord (discord.com)
- X / Twitter (x.com)
- その他テキストエリアやcontentEditableを使うサイト全般

## カスタマイズ

### トースト通知を無効にしたい場合

`content.js` の `showBlockedToast` 関数を呼び出している箇所をコメントアウトしてください。

### 特定サイトでの挙動を調整したい場合

`content.js` の `SEND_BUTTON_SELECTORS` にサイト固有のセレクタを追加できます。

## トラブルシューティング

- **一部サイトで送信できない**: そのサイトが独自のキーイベント処理をしている可能性があります。ポップアップからそのサイトをOFFにしてください。
- **日本語入力中にEnterが効かない**: IME変換中のEnterは自動的にスルーするようになっていますが、問題がある場合はissueをください。
- **設定が反映されない**: ページをリロードしてみてください。
